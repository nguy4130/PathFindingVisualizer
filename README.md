# PathFindingVisualizer

## To get started

Clone this repository to your local machine:

```shell
git clone https://github.com/nguy4130/PathFindingVisualizer.git
```

To open the GUI, run the following command:

```shell
./gradlew run
```

or 

```shell
gradle wrapper run
```
